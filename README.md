# AI2Human Skills

Agent-readable skills for creating campaigns, requesting verification, and reading proof from AI2Human.

## Why This Exists

AI2Human should be callable by agents, not only usable through a web interface.

This repo contains the public skill surfaces that allow an agent to:

- create a campaign draft
- preview funding and winner rules
- request human verification
- request B20 proof-gated issuance support
- read public reports
- test flows through OpenClaw-compatible instructions
